🌍 Green Standards Dashboard

Real-time monitoring system for 100+ schools across 10+ countries

📋 THE CHALLENGE

Goal: Aga Khan Schools network needed real-time tracking of sustainability metrics across 100+ geographically distributed schools in 10+ countries.

Key Questions:

  • How to visualize 100+ data points across regions?
  • How to track 7 environmental metrics per school?
  • How to monitor 8 green standards compliance?
  • How to scale to 1000+ schools?

💡 THE SOLUTION

Built end-to-end web-based monitoring system with:

  • Interactive Geospatial Map: Leaflet.js visualization of 100+ school locations with real-time status indicators
  • Real-time Analytics Dashboard: KPI stat cards with 5+ interactive chart types (Chart.js)
  • Compliance Monitoring: Track 8 green standards across all schools with visual progress bars
  • School Profiles: Individual school data with historical tracking and benchmarking
  • Data Entry Form: Real-time metric updates with validation

Deployment Scale:

## 🎨 Customization | Anpassung | Personnalisation

### Adding New Languages | Neue Sprachen hinzufügen | Ajouter de Nouvelles Langues

1. Create new HTML files: `filename-xx.html` (where xx is language code)
2. Translate all content
3. Update language selector in header navigation
4. Add language to SUPPORTED_LANGUAGES in `language-selector.js`
5. Update sitemap.xml with new language URLs

### Updating Content | Inhalt aktualisieren | Mise à Jour du Contenu

Remember to update **all language versions** when making content changes:
- English: `index.html`, `generic.html`
- French: `index-fr.html`, `generic-fr.html`
- German: `index-de.html`, `generic-de.html`

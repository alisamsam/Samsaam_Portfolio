## 🔄 How Language Detection Works | Wie Spracherkennung funktioniert | Comment fonctionne la détection linguistique

### Step 1: Browser Detection | Schritt 1: Browsererkennung | Étape 1: Détection du navigateur
```javascript
// Automatically detects browser language
const browserLang = navigator.language.split('-')[0].toLowerCase();
// Example: 'de', 'fr', 'en'
```

### Step 2: User Preference Storage | Schritt 2: Speicherung der Benutzereinstellung | Étape 2: Stockage des préférences utilisateur
```javascript
// Saves language choice in localStorage
localStorage.setItem('portfolio-language', 'de');
```

### Step 3: URL-Based Language | Schritt 3: URL-basierte Sprache | Étape 3: Langue basée sur l'URL
```
index.html        // English
index-fr.html     // French  
index-de.html     // German
```

## 🚀 How to Use | Wie man es benutzt | Comment l'utiliser

### For First-Time Visitors:
1. Visit the portfolio
2. Browser automatically detects your language
3. Portfolio loads in your preferred language
4. Your choice is saved for future visits

### To Change Language:
1. Look for language buttons in the header: 🇬🇧 🇫🇷 🇩🇪
2. Click your preferred language
3. Page instantly loads in that language
4. Your preference is saved

### To Reset Language:
1. Clear browser cache/localStorage
2. Or manually click a different language button

## 📝 HTML Tags You'll See

```html
<h1>Big Heading</h1>           ← Large title
<h2>Medium Heading</h2>        ← Section title  
<p>Paragraph text</p>          ← Normal text
<a href="#">Link</a>           ← Clickable link
<img src="image.jpg">          ← Image
<ul><li>List item</li></ul>    ← Bullet list
```

**Remember:** Every tag that opens `<tag>` must close `</tag>`

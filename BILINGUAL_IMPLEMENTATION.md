---

## 🎯 How It Works:

1. **User visits your website** → Sees English version by default (index.html)
2. **User clicks 🇫🇷 FR button** → Switches to French version (index-fr.html)
3. **User clicks 🇬🇧 EN button** → Switches back to English version
4. **All content translates** → Every section changes to selected language

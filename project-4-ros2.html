🦾 ROS2 Autonomous Navigation Robot

Team Project - 8 Members | Robotics & AI Integration

🤖 PROJECT OVERVIEW

Team Size: 8 Members | Context: M2 Advanced Mechatronics | Status: 🔄 Active Development

Design and build autonomous robot with ROS2-based navigation system, computer vision, and obstacle avoidance.

🎯 OBJECTIVES

  • Autonomous navigation in unknown environments
  • Real-time obstacle detection and avoidance
  • Path planning and optimization
  • SLAM (Simultaneous Localization and Mapping)
  • Multi-sensor fusion

🛠️ TECHNOLOGY STACK

Software
  • ✓ ROS 2 (Middleware)
  • ✓ C++ (Core nodes)
  • ✓ Python (Scripts)
  • ✓ OpenCV (Vision)
Hardware & Simulation
  • ✓ Gazebo (Simulation)
  • ✓ Nav2 (Navigation)
  • ✓ SolidWorks (Design)
  • ✓ SLAM Toolbox

🚀 KEY FEATURES

  • Autonomous path planning using A* and Dijkstra algorithms
  • Real-time SLAM with map generation
  • Dynamic obstacle avoidance
  • Multi-sensor data fusion (LiDAR, Camera, IMU)
  • Gazebo simulation environment
